# Customer Support Ticket System

A web-based Customer Support Ticket System developed using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL. The application allows customers to register, log in, create support tickets, track ticket status, and communicate through a live chat system. Administrators can manage tickets, update their status, and view analytics.

## Features

- User Registration & Login
- Customer Dashboard
- Create Support Tickets
- View My Tickets
- Admin Panel
- Ticket Status Management
- Analytics Dashboard
- Knowledge Base
- Live Chat with Bot Reply
- MySQL Database Integration

## Technologies Used

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/customer-support-ticket-system.git
```

2. Install dependencies

```bash
npm install
```

3. Configure MySQL and create the required database.

4. Start the server

```bash
node server.js
```

5. Open your browser

```
http://localhost:5000
```
